## 🔖 Sobre
######  Este Projeto faz parte de um evento do qual tive a oportunidade de participar, onde foi sugerido a **Maratona-Discovery** da Rocketseat, foi  uma maratona de estudos sobre Programação com foco em Desenvolvimento Web, onde o objetivo principal é mostrar como é simples desenvolver projetos web. Nesta edição desenvolveremos um projeto denominado dev.finance$, um site responsivo de controle financeiro.

###### Nesta Maratona pude aprender mais sobre as seguintes técnologias e partilhar conhecimentos com diversos outros desenvolvedores com objetivo principal de criação deste sistema: **dev.Finance$** que é uma aplicação Web e Mobile possibilitando aos usuários realizar um controle de suas transações financeiras, obtendo uma gestão de seus ganhos e gastos pessoais.

---
## 💻 Projeto

##### O **dev.finance$** é uma aplicação de controle financeiro, onde é possível cadastrar e excluir transações e ver o saldo de entrada e saída 💰

## 📁 Como Baixar o Projeto
```bash
  # Clonar o repósitorio
  $ git clone https://github.com/alxlima/DevFinance.git
  
  # Entrar no diretório
  $ cd c:/maratona-discover

  # Instalar as dependências
  $ yarn install
  
  # Iniciar o Projeto
  $ yarn start
```
